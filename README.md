# 🧮 Projeto IMC – Spring Boot
## 📌 Objetivo do projeto
Dando continuidade à aplicação que venho desenvolvendo em **Spring Boot**, este projeto tem como objetivo aplicar, de forma prática, os conceitos de **arquitetura de software MVC (Model–View–Controller)**.

O foco inicial do projeto é construir uma **base bem estruturada, organizada e alinhada com boas práticas de desenvolvimento backend**, garantindo que a **regra de negócio seja implementada de forma correta e sustentável** à medida que a aplicação evolui.
## 🌱 Estado atual do projeto
Atualmente, o projeto contém apenas as camadas de **Controller** e **DTO**, sem regras de negócio ou persistência de dados.

**plantando a semente** para uma aplicação mais completa, que será evoluída gradualmente.

## 🚀 Próximos passos
- Implementação da camada **Model**
- Implementação da camada **Repository**
- Implementação da camada **Service**
